Sobre el Asistente de Hardening

El Asistente Interactivo de Hardening es una herramienta informativa que analiza la configuración de seguridad del sistema Windows y proporciona recomendaciones detalladas para mejorarla. No realiza cambios automáticos en el sistema, solo ofrece orientación paso a paso basada en buenas prácticas.

Datos que analiza:
  • Información del sistema operativo y hardware (CPU, RAM, red, discos).
  • Usuarios locales y cuentas con privilegios elevados.
  • Puertos abiertos y servicios activos.
  • Estado de seguridad: Firewall, BitLocker, Defender, RDP, SMBv1, etc.
Funcionamiento:
  1. Recolecta información del sistema mediante psutil y PowerShell.
  2. Ejecuta comprobaciones de seguridad (B-001 a B-009) del 30/10/2025.
  3. Genera un puntaje global y muestra resultados visuales.
  4. Incluye guías detalladas para cada hallazgo detectado.
Beneficios:
  • Visión clara del estado de seguridad del sistema.
  • Recomendaciones guiadas sin alterar la configuración.
  • Informes legibles en HTML exportables.
  • Ideal para estudiantes y ingenieros de sistemas.

¿Cómo funciona la recopilación de datos?

VirusHunter combina tecnologías de PowerShell y Python (psutil) para analizar tu sistema sin comprometer tu privacidad. Los datos recolectados son de diagnóstico técnico y no personales.

Análisis del hardware

Mediante psutil, el sistema recopila información sobre CPU, memoria y disco para evaluar el rendimiento general del equipo.

Revisión de seguridad

Con comandos PowerShell, se verifica el estado del firewall, antivirus, usuarios y protocolos del sistema operativo.

Generación de informes

La información obtenida se organiza automáticamente en un archivo JSON, que resume el estado de seguridad del sistema.

Sobre nuestro proyecto
